# Create Inventory App

## Use JSON Data
Create Data.js like this :

```javascript

[
    "product" : {
        name: "productname",
        detail: "detailProduct",
        price: "price",
        stock: "jumlahStock"
    },
    "order" : {
        name: "productname",
        total: "totalProduct",
        price: "price",
    } 
]
```
Show data product and calculate product stock with order, stock - order = total stock item
